BFS (rel, root, funItem, retRevPath)
Breadth first search over the relation. Do a breadth first search from root over the relation. Source

Parameters:

  • rel Relation
  • root The root node to the search.
  • funItem Function executed over the elements. It has the signature funItem(elem, topIndex).
  • retRevPath Save the sequence nodes.

Returns:

    return the path if retRevPath is true.

Usage:

    rel:BFS("nRoot", function(elem, topIndex)
  print(elem, topIndex)
  return true
end)
